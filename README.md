# screenshot-todo-skill

スクリーンショット画像から課題名・締切日・時刻・予定を抽出し、ToDoリスト形式で表示する静的Webアプリです。

## 起動方法

```bash
python3 -m http.server 8080
```

ブラウザで `http://localhost:8080/` を開きます。

## 機能

- PNG、JPG、WEBPなどの画像アップロード
- アップロード画像のプレビュー
- Tesseract.js による日本語 + 英語OCR
- OCR前の画像拡大、グレースケール化、コントラスト補正
- `6/20`、`6月20日`、`2026-06-20`、`2026年6月20日` 形式の日付検出
- `23:59` 形式の時刻検出
- 日付の前後2〜3行を使った課題名・予定名の推定
- 「提出」「締切」「期限」「deadline」「due」などのキーワード優先
- 日付がない課題候補は `期限不明` として表示
- 検出したToDoを時系列順に表示

## ファイル構成

- `index.html`: アプリ画面
- `styles.css`: レイアウトと表示スタイル
- `script.js`: OCR前処理、OCR実行、情報抽出、ToDo描画
- `skill.md`: AI Skillとしての目的、入力、処理、出力、例

## 注意

OCRには Tesseract.js CDN を使用します。抽出を実行するブラウザ環境からCDNへアクセスできる必要があります。
